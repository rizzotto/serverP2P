# Client CLI

Into the client folder, run 

```shell
$ npm install
$ npm link
```
then `client` command is ready to use

## COMMANDS

Commands available

```shell
$ client -h
$ client post-resources
$ client list-all-resources
$ client get-specific-resource
$ client test
```

# Server

Into the server folder, run

```shell
$ npm install
% npm run dev
```
After this, the server is running
